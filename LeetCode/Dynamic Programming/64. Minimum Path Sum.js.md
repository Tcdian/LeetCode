## [64. Minimum Path Sum](https://leetcode.com/problems/minimum-path-sum/)

###

Given a `m x n` `grid` filled with non-negative numbers, find a path from top left to bottom right, which minimizes the sum of all numbers along its path.

**Note**: You can only move either down or right at any point in time.

#### Example 1:

<img src="https://assets.leetcode.com/uploads/2020/11/05/minpath.jpg" width="242" />

```
Input: grid = [[1,3,1],[1,5,1],[4,2,1]]
Output: 7
Explanation: Because the path 1 → 3 → 1 → 1 → 1 minimizes the sum.
```

#### Example 2:

```
Input: grid = [[1,2,3],[4,5,6]]
Output: 12
```

#### Constraints:

-   `m == grid.length`
-   `n == grid[i].length`
-   `1 <= m, n <= 200`
-   `0 <= grid[i][j] <= 100`

#

### 方法一: 动态规划（Dynamic Programming）

#### 代码

```javascript
/**
 * @param {number[][]} grid
 * @return {number}
 */
var minPathSum = function (grid) {
    const dp = Array.from(new Array(grid.length), () => new Array(grid[0].length));
    for (let i = 0; i < grid.length; i++) {
        for (let j = 0; j < grid[0].length; j++) {
            if (i === 0 && j === 0) {
                dp[i][j] = grid[i][j];
            } else if (i === 0) {
                dp[i][j] = dp[i][j - 1] + grid[i][j];
            } else if (j === 0) {
                dp[i][j] = dp[i - 1][j] + grid[i][j];
            } else {
                dp[i][j] = Math.min(dp[i - 1][j], dp[i][j - 1]) + grid[i][j];
            }
        }
    }
    return dp[grid.length - 1][grid[0].length - 1];
};
```

#### 复杂度

-   时间复杂度: _O_ (m \* n)，其中 `m` 和 `n` 分别是 `grid` 的行数和列数。

-   空间复杂度: _O_ (m \* n)。
